The css pseudo-classes allow you to style the dynamic states of an element such as hover, active and focus state,as well as elements that are existing in th document tree but can't be trageted via the use of other selectors without adding ant IDs or classes to them, for ecxample, targeting the first or last chaild element.

A pseudo-class starts with a colon(:). Its syntax can be given with:

select:pseudo-classes {prperty: value;}
